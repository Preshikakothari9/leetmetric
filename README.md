# LeetMetric

A simple and interactive web application that fetches and displays LeetCode user statistics.

## Features
- Fetch LeetCode user data using API
- Display Easy, Medium, Hard problem progress
- Circular progress visualization
- Clean and responsive UI
- Error handling for invalid users / API issues

## Tech Stack
- HTML
- CSS
- JavaScript

## How to Use
1. Enter your LeetCode username
2. Click on Search
3. View your problem-solving stats instantly
